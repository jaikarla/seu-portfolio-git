# Portfólio Git - Desenvolvimento de Software

Repósitorio criado para a atividade EI1 da disciplina de Dsenvolvimento de Software, 2026.1.

O objetivo desse repositório é demonstrar o uso de Git de forma prática, com commits semânticos, uso de branches, Pull Requests e resolução de conflitos.

## Estrutura do repositório

```
seu-portfolio-git/
├── README.md
├── reflexao.md
├── exercicios/
│   ├── ex01-commits/
│   │   ├── output/
│   │   ├── ex1.cpp
│   │   ├── hello.cpp
│   │   ├── math.cpp
│   │   ├── oi.txt
│   │   └── teste.cpp      
│   ├── ex02-branches/
│   │   └── 1.txt
│   └── ex03-conflito/
│   │   ├── registro.md
│   │   └── texto.txt
└── projeto/
    └── hello-world/
        └── hello2.cpp
```

`exercicios/`: contém os arquivos utilizados para o exercício.
- ex01-commits: exemplos de commits semânticos seguindo o padrão Conventional Commits.
- ex02-branches: prática de criação e uso de branches.
- ex03-conflito: demonstração de um conflito de merge e seu registro.

`projeto/`: um pequeno projeto "hello world" para os exercícios de branches e commits.
- hello-world: programa simples usado para demonstrar alterações no código e histórico de commits.

## Tecnologias utilizadas
- Git
- GitHub
- C++

## Objetivo do exercício

Este repositório busca demonstrar a compreensão dos seguintes conceitos:

- uso de commits semânticos
- organização de trabalho com branches
- uso de Pull Requests
- resolução de conflitos de merge
- reflexão sobre o processo de aprendizagem do Git
